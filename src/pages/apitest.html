    Haetaan nappia painamalla kaikki käyttäjät

    GET http://localhost:3000/api/users

    GET

    Hae käyttäjän tiedot painamalla alla olevaa nappia. Tämä hakee tiedot sekä luo taulukon tbody osan. Jokaiselle riville luodaan nappulat. Info hakee yksittäisen rivin tiedot ja avaa alert-popupin. Delete poistaa yksittäisen rivin ja varmistaa tapahtuman confirm() kun reitti on valmiina. metodilla. Info on kotitehtävä ja Delete toiminto lisätään myöhemmin

    User Name Email Info Delete ID
